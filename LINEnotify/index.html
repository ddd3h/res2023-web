    毎日自動バックアップを行い完了時にLINEに通知させる

    ファイル名「LINE.py」で以下のコードを入力します。

    line_notify_tokenにはLINE API Tokenを取得して入力してください。

    import requests
import sys

def notify(notification_message):
    """
    LINEに通知する
    """
    line_notify_token = ''
    line_notify_api = 'https://notify-api.line.me/api/notify'
    headers = {'Authorization': f'Bearer {line_notify_token}'}
    data = {'message': notification_message}
    requests.post(line_notify_api, headers = headers, data = data)


if __name__ == '__main__':
    args = sys.argv
    if 2 <= len(args):
        notify(args[1])

    これにより、次のコマンドで関数を呼び出して通知させることができます。

    import LINE
LINE.notify("[Anna計算機] バックアップが完了しました。")

    また、シェルコマンド上でも使用することができます。

    そのため、nohupを使用する長時間かかる解析であっても利用できます。以下に例を書いておきます。

    python LINE.py "解析終了"
./LINE.py "解析終了"
